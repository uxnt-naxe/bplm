## BPLM - 你好世界
```go
plug "stdio.h"
func main()
     var x = 2021
     print("Hello, World" + " - " + x)
end

```
打印输出：

```
Hello, World - 2021
```

## BPLM - 算法

```go
plug "stdio.h"
func main()

```

## BPLM - 判断
```go
plug "stdio.h"
func main()
     var x = 6
     if(x > 5)
          printf("Well done!")
     end
end
```
打印输出：
```
Well done!
```
